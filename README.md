# Домашнє завдання №5 — Git базовий робочий процес

## Проєкт

Даний проєкт — це шаблон сайту для пошуку роботи на **HTML/CSS**, зв'язаний з Git:

- `index.html` — головна сторінка.
- `index.html1` — шаблон для майбутніх змін головної сторінки та поділ її на блоки.
- `style.css` — спільний стиль для усіх сторінок.

## Що зроблено (план робочого процесу)

1. Ініціалізовано репозиторій.
2. Додано базову HTML/CSS структуру.
3. Оформлено зміст сайту.
4. Підготовлено звіт про пройдені команди Git.
5. Описано особливості проєкту в цьому README.

## Команди Git, які використано

- `git init`
- `git add`
- `git commit`
- `git remote add origin <URL>`
- `git push -u origin master`
- `git status`
- `git log`
